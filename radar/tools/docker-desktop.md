[Docker Desktop](https://www.docker.com/products/docker-desktop) is a tool for
MacOS and Windows that allows to run Docker containers locally.

Since Docker Desktop license is not compatible with our policies, you are not
allowed to use it in our organization.

Use [Rancher Desktop](https://rancherdesktop.io/) instead.
