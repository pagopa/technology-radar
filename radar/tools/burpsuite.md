Burp Suite is a software security application used for penetration testing of web applications. The software is developed by the company PortSwigger. The suite includes tools such as a proxy server (Burp Proxy), an indexing robot (Burp Spider), an intrusion tool (Burp Intruder), a vulnerability scanner (Burp Scanner) and an HTTP repeater (Burp Repeater).
