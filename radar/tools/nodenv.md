[nodenv](https://github.com/nodenv/nodenv) is a tool that allows you to install and switch between multiple Node.js versions seamlessly.

It offers extended features to its [nvm](https://github.com/nvm-sh/nvm) competitor, such as [autoswitching](https://github.com/nodenv/nodenv/wiki/Why-nodenv%3F#auto-switching), also turns out to be much more stable and [faster](https://github.com/nodenv/nodenv/wiki/Why-nodenv%3F#performance).

## Use cases

- Use the right version of Node.js across projects
