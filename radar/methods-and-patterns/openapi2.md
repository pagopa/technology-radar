The [OpenAPI Specification (OAS) v2](https://swagger.io/specification/v2), also
known as Swagger used to be the standard version to document and automate API
specifications.

As [OpenAPI Specification (OAS) v3](openapi3.html) became publicly adopted, we
no longer encourage the usage of OpenAPI v2.
