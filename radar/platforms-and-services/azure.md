Azure is Microsoft's public cloud computing platform. It provides a range of
cloud services, including those for computing, analytics, storage, and
networking. Users can pick and choose from these services to develop and scale
new applications, or run existing applications in the public cloud. Azure
supports a range of operating systems, databases, tools, programming languages
and frameworks, including both Microsoft-specific and third-party software and
systems.
